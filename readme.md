### THE PLAN

An idle game inspired by the iconic bouncing DVD logo \
The logos bouncing earns you points to buy upgrades that allows for more points and more logos. \
\
The green shop button increases speed \
The red shop button increases points gained per bounce and click \
The purple button adds 1 dvd logo \
\
The numbers above the shop icons are the prices \
The number in the top left corner is your wallet
